# NClientT

A cross-platform **desktop** port of [NClientV3](https://github.com/maxwai/NClientV3),
built with **Tauri 2** (Rust backend) + **Vue 3** (TypeScript frontend).

> вҡ пёҸ This is an unofficial, hobbyist client and is for personal, lawful use only.
> Respect the source site's Terms of Service and your local laws. The project
> focuses purely on the technical port (networking, downloads, auth, etc.).

## What's ported from NClientV3

| NClientV3 (Android)                       | NClientT (desktop)                                |
|-------------------------------------------|---------------------------------------------------|
| `OkHttpClient` + `CustomCookieJar`        | `reqwest` + `reqwest_cookie_store` (JSON jar)     |
| `ApiAuthInterceptor` (UA + `Key <api>`)   | `http.rs` вҖ” same UA + `Authorization: Key <key>`  |
| `InspectorV3` (browse/search/random)      | `api.rs` вҖ” same v2 endpoints                      |
| `GalleryDownloaderV2` + `DownloadQueue`   | `downloader.rs` вҖ” concurrent, pause/cancel/resume |
| `CookieInterceptor` + `CFTokenView` (CF)  | `cloudflare.rs` вҖ” webview window + cf_clearance   |
| `async/database/Queries` (SQLite)         | `db.rs` вҖ” favorites/history/tags/local/downloads  |
| `ApiAuthInterceptor` + `AuthStore`        | `config.rs::AuthCredentials` + `auth_*` cmds      |
| PDF / ZIP export                          | `export.rs` (`lopdf` + `zip`)                     |
| Activities / fragments                    | Vue 3 views + Pinia stores                        |

### Features

- рҹҸ  **Home** вҖ” browse recent / popular (day / week / month / all)
- рҹ”Қ **Search** вҖ” by query + tag filters (include / exclude) + language + page-range
- рҹҺІ **Random** gallery
- рҹ“– **In-app reader** вҖ” fit width / height / 1:1, RTL, keyboard nav
- в¬Ү **Download manager** вҖ” queue, parallel pages, pause / resume / cancel, progress events
- вҳ… **Favorites** вҖ” local (DB) and online (requires API key)
- рҹ•‘ **History** вҖ” auto-recorded reads
- рҹ“Ғ **Local library** вҖ” scans the download dir, reads gallery `.nomedia` metadata
- рҹҸ· **Tags** вҖ” cached, searchable, mark include/exclude
- рҹҢҗ **Cloudflare** вҖ” challenge solved in an embedded webview, `cf_clearance` reused by the HTTP client
- рҹ”‘ **API key auth** вҖ” sent as `Authorization: Key <key>` on every API call
- рҹҚӘ **Persistent cookies** вҖ” survive restarts (so do CF tokens)
- вҡҷ **Settings** вҖ” mirror, User-Agent, timeouts, columns, zoom, RTL, download dir, ...
- рҹ“Ұ **Export** вҖ” convert a downloaded gallery to PDF or ZIP
- рҹ–Ҙ **Cross-platform** вҖ” Windows first; macOS / Linux ready (Tauri 2)

## Windows вҖ” first-time build setup

Tauri on Windows needs **Rust** and the **MSVC C++ build tools** plus **WebView2**.

### 1. Install Visual Studio Build Tools (MSVC)

Download **"Build Tools for Visual Studio 2022"**:
<https://visualstudio.microsoft.com/visual-cpp-build-tools/>

In the installer, check **"Desktop development with C++"**, which pulls in:
- MSVC v143 build tools
- Windows 11 SDK (or Windows 10 SDK)
- C++ CMake tools

### 2. Install Rust

Install via **rustup** (the official installer):
<https://www.rust-lang.org/tools/install>

In a **new** PowerShell / cmd window verify:

```powershell
rustc --version
cargo --version
```

> The default `stable-x86_64-pc-windows-msvc` toolchain is what we want.

### 3. Confirm WebView2 runtime

WebView2 is preinstalled on Windows 10/11. If missing, grab the Evergreen
bootstrapper from <https://developer.microsoft.com/microsoft-edge/webview2/>.

### 4. Build & run

```powershell
cd Y:\NClientT

# Install JS deps (Vue, Tauri CLI, Pinia, ...)
npm install

# Generate all icon sizes (PNG/ICO/ICNS) from the source PNG вҖ” no deps needed
npm run icon

# Dev: launches the Vite dev server + Rust shell with hot reload
npm run tauri:dev

# Production build вҶ’ .msi / .exe installer under src-tauri/target/release/bundle/
npm run tauri:build
```

The first `cargo` build downloads and compiles all crates (5вҖ“15 min depending
on machine). Subsequent builds are fast.

## macOS / Linux

The same Rust + npm steps work; only the prerequisites differ:

- **macOS**: `xcode-select --install` then rustup. `npm run tauri:build` вҶ’ `.app` / `.dmg`.
- **Linux**: install `webkit2gtk`, `libssl-dev`, `librsvg2`, etc. (see
  <https://tauri.app/start/prerequisites/>), then rustup + npm.

## API key (optional)

Without an API key the client behaves like NClientV3 in logged-out mode: browse,
search, read, and download all work. Online favorites and comments require a key.
Add it under **Settings вҶ’ API Key Authentication**.

## Cloudflare

If the source site is behind Cloudflare, the home/search pages will surface a
"Cloudflare verification required" banner. Click **Solve now** (or open
**Settings вҶ’ Cloudflare вҶ’ Solve**). A small webview opens, you solve the
challenge like a normal browser, and the resulting `cf_clearance` cookie is
captured and reused by the Rust HTTP client for all subsequent requests вҖ”
including downloads.
